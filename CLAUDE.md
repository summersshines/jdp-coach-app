# JDP Coach App — Ryde Parramatta

## Overview
Single-file HTML/CSS/JS coach app for Anthony Summers' Junior Development Program at Ryde Parramatta Golf Club. Hosted on GitHub Pages. Shared Supabase backend with St Michael's app.

## Live URL
summersshines.github.io/jdp-coach-app/

## Tech Stack
- Single-file HTML/CSS/JS (no frameworks, no build step)
- GitHub Pages deployment
- Supabase (shared backend — PostgreSQL, REST API via JS client)
- Tested on mobile after each deploy

## Key Differences from St Michael's App
- Groups: Beginner, Tier 3, Tier 2, Tier 1
- Supabase data uses no prefix (STM uses stm_ prefix)
- Drill videos shared under drill_videos key (same as STM)
- Different colour scheme and branding
- Different function names and structure — do not copy STM code directly

## Working Rules
- One change at a time
- Always read index.html first and propose before editing
- No bundling of multiple changes
- Mobile-first — changes must work on mobile
- Commit message should describe what changed and why
